PDF Saver

Save any article as PDF from Safari — no app install needed

Step 1 — Install the bookmarklet

iOS Safari doesn't allow saving javascript: links by tapping. Use this 3-step trick instead:

1 Tap Share → Add Bookmark. Name it Save as PDF and save it anywhere.
2 Tap the button below to copy the bookmarklet code to your clipboard.
3 Open Bookmarks, find Save as PDF, swipe left → Edit. Tap the URL field, select all & delete it, then paste the copied code. Tap Done.

On desktop: drag this link to your bookmarks bar instead → Save as PDF

Step 2 — Use it on any article

  1. Open an article in Safari.
  2. Tap the address bar, then Bookmarks, then Save as PDF.
  3. A clean version of the article opens in a new tab.
  4. Tap the Share button → Print.
  5. Pinch outward on the page preview thumbnail.
  6. Tap Share again → Save to Files → Save.
iOS Print → PDF trick: When the print sheet appears, pinch outward on the small page preview. This converts the print job into a PDF file that you can save anywhere.

If a pop-up is blocked

Safari may block the new tab on the first use. Tap the Pop-up Blocked notice in the address bar area and choose Allow, then tap the bookmark again.
