# safari-pdf-addin
An add-in to convert a web-page to pdf on Safari on the phone
The problem statement is as follows: When I find an interesting article that I am reading on Safari (on an iphone) and I want to share it with a friend, then I turn on reader mode and share as PDF. But reader mode has a lot of bugs and is unreliable to turn on. I want to create an add-in for safari on the phone that can help convert the page to pdf
